Resource: Embed

Methods

HTTP-Method
GET
URI:
http://embedit.me/embedit/?url={url}&height={height}&width={width}&api_key={api_key}&frog_key={frog_key}
Description:
Content-Type should be set to application/json or application/xml
Authentication
Request Example:
http://embedit.me/embedit/?url=http://youtube.com/xyz&height=500&width=500&api_key=my_fancy_api_key&frog_key=my_fancy_frog_key
